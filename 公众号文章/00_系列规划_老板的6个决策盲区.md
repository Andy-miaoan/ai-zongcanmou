# AI总参谋公众号文章 · 系列规划

> 系列名：**老板的6个决策盲区**
> IP人设：Andy，企业咨询14年，帮中小企业老板把决策从"拍脑袋"变成"有框架"
> 节奏：每周2-3篇，每篇拆一个盲区，6篇为一个完整系列
> 风格：反思型，70-80%工具人内容+≤8%产品植入，真实案例+方法论

---

## 系列定位

**一句话：** 不卖产品，卖"原来我一直在犯这个错"的认知冲击。每篇文章拆一个老板最深的决策盲区——先让你认账，再给方法论，最后自然带出"这个方法论我做成了工具"。

**连续性设计：**
- 每篇末尾有明确的"下篇预告"——预告具体主题+核心价值
- 每篇开头用1-2句话衔接上一篇
- 六篇结束后，读者自然想要全套工具
- 读者加微信后不需要再解释产品是什么——六篇文章已经完整展示了六个模块的价值

## 每篇文章统一结构

```
开头故事（具体人物+具体细节+痛点台词）
  → 痛点放大（数据/案例）
    → 认知翻转（反常识结论）
      → 【反思停留】150-200字
        → 方法拆解（2-5个模块）
          → 划重点（4条金句）
            → 下篇预告（具体主题+核心价值）
              → 互动引导（评论区）
                → 关注引导+微信CTA
```

## 产品植入原则（≤8%）

- 只在"方法拆解完成后"出现一次
- 不描述产品功能，只说"这个方法论我做成了工具"
- 一句话带过：做了什么、能帮什么、怎么获取
- 其余76%+是纯粹的干货内容
